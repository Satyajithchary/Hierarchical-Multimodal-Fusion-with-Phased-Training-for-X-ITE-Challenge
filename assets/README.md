# assets/

Place the following figures extracted from the paper in this directory:

| Filename | Source in paper | Description |
|---|---|---|
| `architecture.png` | Fig. 1 | Hierarchical Multimodal Fusion Architecture diagram |
| `training_curves.png` | Fig. 2 | Training & validation loss and accuracy curves |
| `evaluation_metrics.png` | Fig. 3 | ROC curve and Precision-Recall curve |

## How to extract figures from the paper PDF

1. Open the paper PDF in any PDF viewer.
2. Use **Edit → Take Screenshot** (macOS Preview) or **Snipping Tool** (Windows) to crop each figure.
3. Save as PNG at ≥ 150 DPI and name according to the table above.

Alternatively, if you have the original figure files from the LaTeX source, copy them directly here.

> The README references these files as relative paths (`assets/*.png`), so they must be committed alongside the code for the images to render on GitHub.
